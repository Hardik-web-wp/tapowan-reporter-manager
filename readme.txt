TAPOWAN Reporter Manager is a custom WordPress plugin for reporter applications, approvals, Press ID cards, appointment letters, verification, and membership management.

== Description ==
This plugin provides a foundation for a TAPOWAN reporter registration and verification system.

Shortcodes:
- [reporter_apply]
- [reporter_dashboard]
- [member_verification]
- [approved_reporters]

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/
2. Activate the plugin from the WordPress admin
3. Create pages and add the shortcodes
